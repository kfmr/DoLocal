# DoLocal


### Projeto 
O DoLocal foi desenvolvido com base no projeto Ecoleta durante a Next Level Week ofertado pela Rocketseat. <br>
A proposta DoLocal é um repositório de pequenos comércios locais. 


## Tecnologias:
- HTML,CSS e Javascript
- Node.js
- Express
- Nunjucks
- SQLite
